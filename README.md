# 网易云音乐Web客户端(改进版) - An improved version of Netease Cloud Music web client

Forked from https://github.com/sl1673495/vue-netease-music. See features for a list of improvements.

Credit to ssh (https://github.com/sl1673495)

### Demo

http://m.piekill.com

### Features

- [x] 邮箱登录 (improvement)
- [x] 云盘音乐 (improvement)
- [x] 私人FM (improvement)
- [x] 喜欢音乐 (improvement)
- [x] 每日推荐 (improvement)
- [x] mv 页 (original)
- [x] cd 页 (original)
- [x] 搜索建议 (original)
- [x] 搜索详情 (original)
- [x] 播放 (original)
- [x] 发现页 (original)
- [x] 播放列表 (original)
- [x] 播放记录 (original)
- [x] 全部歌单 (original)
- [x] 歌单详情 (original)
- [x] 最新音乐 (original)
- [x] 主题换肤功能 (original)
